## Getting Started

To get a local copy up and running follow these simple example steps.

- git clone 

- cd search-engine


### Prerequisites
To start please have the following running
- Ruby
- Rails
- PostgresSQL


### Setup

Install gems with:

```
bundle install
```

Setup database with:

```
rails db:create
rails db:migrate
```

Start server with:

```
rails server
```

Open ```http://localhost:3000/``` in your browser.


### Run tests

Install npm with:

```
npm i
```

Install rspec with:

```
bundle install
```

and

```
rails generate rspec:install
```

run the test with:

```
rspec spec
```
